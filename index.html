How To Edit

Add or edit recipes in the recipes/ folder. Each recipe is a Markdown file with front matter for category, ingredients, tags, pantry labels, and source. Then run the R build script to regenerate the site data and recipe pages. 

Rscript scripts/build-recipes.R

Pantry items live in data/pantry.md. Calendar meals come from each recipe's dates_cooked field.
